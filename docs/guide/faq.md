# 常见问题

## API Key 填了还是 401？

先查 Key。

- 是否复制完整。
- 前后是否有空格或换行。
- 是否来自正确的服务商。
- 是否过期、撤销或额度用完。
- 是否填到了正确字段。

详见 [API Key 教程](/guide/api-key)。

## Base URL 要不要带 `/v1`？

看工具页示例。

有些工具要你带 `/v1`。

有些工具会自动加 `/v1`。

如果报 404，优先检查这里。

详见 [Base URL 教程](/guide/base-url)。

## 为什么提示模型不存在？

通常是模型名不对。

- 拼写是否完全一致。
- 大小写是否一致。
- 连字符和日期版本是否正确。
- 你的 Key 是否有权限使用这个模型。
- 服务商是否还支持这个模型。

能从工具模型列表选择时，优先从列表里选。

## Provider 应该选哪个？

选当前工具页要求的 Provider。

不要只看服务商名称。

重点看接口格式：

- OpenAI Chat
- OpenAI Responses
- Anthropic Messages

选错后，可能会出现 404、模型不存在或接口格式错误。

## 为什么工具里能保存，但请求失败？

保存只代表配置写进去了。

请求是否成功，还要看接口是否能跑通。

检查：

- 是否已经保存。
- 是否需要重启或重新加载。
- Base URL 是否正确。
- API Key 是否有效。
- 模型名称是否可用。
- Provider 是否选对。

详见 [验证配置教程](/guide/verify)。

## image2 能不能填到代码助手里？

通常不能直接当聊天模型填。

image2 走图片生成或图片编辑接口。

代码助手通常走 OpenAI Chat、OpenAI Responses 或 Anthropic Messages。

如果工具页没有写支持图片接口，不要把 image2 当代码助手模型填。
